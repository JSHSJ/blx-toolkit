Modal

There's two things to a modal.

The Modal / Dialog handling

Handled by a custom element 

`blx-modal`
and consists of

  • A button with attribute 'data-action="click:blx-modal#open"' to open the modal
  • A container element with attribute 'data-target="blx-modal.container"' to contain all of the modal
  • An overlay element with attribute 'data-target="blx-modal.overlay"' that sits below the content but on top of the document to help focus on the content.
  • A content container element with attribute 'data-target="blx-modal.content"', that sits centered in the middle (or whereever else you like)
  • An optional, but recommended title with attribute 'data-target="blx-modal.modalTitle"'. This is used to label the dialog.
  • An optional, but recommended button with attribute 'data-target="blx-modal.close"' and 'data-action="click:blx-modal#dismiss"''. This will close the element.

This custom element handles all of the logic, accessibility and niceties, but is completely unstyled, although the animations are included.
